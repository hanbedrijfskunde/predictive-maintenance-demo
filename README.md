# PredictivePulse AI – Digital Twin Demo

Een interactieve webapplicatie die predictief onderhoud simuleert met behulp van een digitale tweeling (digital twin) van industriële machines.

## Doel

Deze applicatie is ontwikkeld als demonstratie voor studenten en professionals in de bedrijfskunde om inzicht te geven in hoe **predictief onderhoud** werkt en welke bedrijfswaarde het oplevert. De simulatie laat zien hoe sensordata van machines real-time kan worden gemonitord en geanalyseerd om ongeplande stilstand te voorkomen.

## Functies

- **Live Digital Twin**: Visualisatie van machinestatus via een real-time digitale tweeling, inclusief invoer- en uitvoerstromen.
- **Health Score Monitor**: Per machine wordt een gezondheidsscore (0–100%) berekend op basis van gesimuleerde slijtageprofielen en resterende levensduur.
- **Vloot Overzicht**: Beheer en monitoring van meerdere machines tegelijk (bijv. Offsetdruk XL-75, Digital Inkjet Pro 8, Vouw/Stans Combo 9).
- **AI Inzichten & Anomalieën**: Automatische detectie van afwijkingen in machinegedrag met bijbehorende aanbevelingen.
- **Aanbevolen Acties**: Op basis van de analyse worden concrete onderhoudsbeslissingen voorgesteld.
- **Impact op Business**: Kwantificering van de bedrijfsvoordelen, waaronder downtime-reductie, besparing op materialen en OEE-verbetering.
- **Reset functie**: Simulatiedata opnieuw instellen voor hergebruik in onderwijssituaties.

## Technologie

De applicatie is gebouwd als een enkelvoudige HTML-pagina (geen externe afhankelijkheden vereist) en is direct te gebruiken via GitHub Pages.
